# プライバシーとデータ保護

- `rule_id`: `codeguard-0-privacy-data-protection`
- 原題: `Privacy & data protection`

- 強力な暗号を実装し、HSTS 付き HTTPS を強制し、証明書ピニングを有効にし、データと匿名性を保護するユーザープライバシー機能を提供する。
- 転送中および保存中のデータには強力で最新の暗号アルゴリズムを用いる。パスワードは確立されたライブラリで安全にハッシュする。
- HTTPS のみを強制し、HTTP Strict Transport Security（HSTS）を実装する。
- CA が侵害されても中間者攻撃を防ぐため、証明書ピニングを実装する。
- 実現可能な範囲でサードパーティの外部コンテンツ読み込みをブロックし、IP アドレスの漏えいを最小化する。
- プライバシーの制限とデータ取り扱い方針についてユーザーに知らせ、透明性を保つ。
- プライバシーに配慮した監査証跡とアクセスログを実装する。
- アカウント列挙を防ぐため、「ユーザー名またはパスワードが無効です」と返す。
- ユーザーごとに一意のソルトを伴う Argon2 または bcrypt を用いる。
- 暗号学的にランダムな ID でセッションをサーバー側に保存する。
